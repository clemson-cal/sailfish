sailfish.setup.Parameter

class sailfish.setup.Parameter(default: Any, about: str, mutable: bool = False)[source]

Bases: NamedTuple

A model parameter: enables runtime configuring of a setup.

Parameter instances are assigned as class variables to Setup sub-classes. The mutable flag should be set to to True if the parameter is logically adjustable either during the run, or can be superseded when a run is restarted.
