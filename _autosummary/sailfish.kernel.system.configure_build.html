sailfish.kernel.system.configure_build

sailfish.kernel.system.configure_build()[source]

Initiate the build_config module-level variable.

The build configuration is presently inferred from a crude check to see whether we’re on Linux or MacOS. This procedure can be easily generalized to accommodate other or more specific platforms, check for hardware or software availability, or read a bulid configuration from a user or site-specified machine file.
